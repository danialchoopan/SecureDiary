# Secure Diary Danial 🔒📔

A secure, modern Android diary application built with Jetpack Compose, focused on privacy and ease of use.
### Features
- **Biometric Security**: Protect your thoughts with fingerprint or face unlock.
- **Pin Authentication**: Alternative secure access via PIN.
- **Rich Diary Entries**: Write and save your daily experiences.
- **Location Support**: Add locations to your entries using Google Maps.
- **Media Support**: Attach images to your diaries using Coil.
- **Persistent Storage**: Uses Room database for fast and reliable data management.
- **Bilingual Support**: Fully supports both English and Farsi (Persian).
- **Modern UI**: Clean and intuitive interface built with Material 3.

### Tech Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Database**: Room
- **Navigation**: Compose Navigation
- **Security**: AndroidX Security Crypto & Biometric
- **Dependency Management**: Gradle Version Catalog
- **Image Loading**: Coil
- **Maps**: Google Maps Compose
